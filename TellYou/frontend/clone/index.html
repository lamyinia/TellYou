设计理念与特色
星空主题
采用深蓝色渐变背景配合动态闪烁星星，营造浪漫而现代的视觉体验，减少长时间使用的视觉疲劳。
毛玻璃效果
大量使用半透明背景和模糊效果，增强层次感和现代感，提升界面的视觉深度。
圆角设计
统一的18px圆角设计语言，营造柔和友好的视觉感受，符合现代UI设计趋势。
交互反馈
丰富的hover效果和过渡动画，提供流畅的用户交互体验，增强操作的反馈感。
色彩搭配
以深蓝为主色调，绿色作为强调色，白色文字确保可读性，形成和谐的色彩体系。
响应式布局
采用Flexbox和Grid布局，确保在不同屏幕尺寸下都能保持良好的显示效果。
🔧
技术实现
前端框架
Vue 3 + TypeScript，采用Composition API，提供类型安全和现代化的开发体验。
桌面应用
Electron框架，支持跨平台部署，提供原生桌面应用体验。
UI组件
Vuetify 3，提供丰富的Material Design组件，快速构建现代化界面。
样式设计
自定义CSS + CSS Grid/Flexbox，实现独特的星空主题和响应式布局。
状态管理
Pinia状态管理，提供响应式的数据管理，支持开发工具调试。
构建工具
Vite + Electron Builder，提供快速的开发体验和高效的打包部署。
